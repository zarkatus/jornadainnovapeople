# Jornada · Ensaios InnovaPeople

Blog editorial da InnovaPeople — ensaios longos sobre design organizacional, engenharia de contexto, retenção de conhecimento e a arquitetura da decisão distribuída.

Produção: https://jornada.innovapeople.com.br

## Stack

- HTML estático puro (sem build)
- Google Fonts: Cormorant Garamond + Inter
- Cloudflare Pages (projeto `jornadainnovapeople`)
- Deploy direto via `wrangler pages deploy` ou via GitHub Actions

## Estrutura

```
.
├── index.html              # Homepage (lista de ensaios)
├── ensaios/
│   └── clock-drift.html    # Ensaio inaugural N.º 01
├── favicon.svg
└── README.md
```

## Posicionamento

> Onde o resultado permanece oculto, o encantamento começa.

Tese central: **encantar é a operação cognitiva precisa de revelar o que sempre esteve ali e ninguém tinha como ver.** A Jornada é a arquitetura desse encantamento aplicada à empresa contemporânea.
